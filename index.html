Pandorita

Cocoa-based pandora.com player using pianolib

Welcome to Pandorita's GitHub Page

Pandorita is a native Mac OS X client for the internet radio service Pandora. It is written entirely in Cocoa, and built to be lightweight, easy to use, and fully functional. Internally it uses the amazing pianobar library to communicate with Pandora's server.

So what can it do?

Pandorita is still in its very early stages, so many features are not yet implemented and some features may have bugs. That said, nearly everything you would want to do with Pandora can already be done in Pandorita. Currently, you can use Pandorita to:

  • Stream tracks from any station.
  • Rate tracks, including ones which have already finished playing.
  • View album artwork for the current song.
  • Create stations by searching for artists and songs, as well as rename or delete existing stations.
  • Enable Growl notifications.
  • Enable global hotkeys.

Last.fm scrobbling is soon to come as well.

Cool! Where can I get it?

You can download the latest version (0.3.1) here. If you are feeling adventurous, you can download and build the absolute latest source from the Github repository, but be warned, this is likely to contain bugs.
